# Event
~~~
HTML DOM events allow JavaScript to register different event handlers on elements in an HTML document.

Events are normally used in combination with functions, and the function will not be executed before the event occurs (such as when a user clicks a button).

part of Dom event
change=The event occurs when the content of a form element, the selection, or the checked state have changed (for <input>, <select>, and <textarea>)

2 Click=The event occurs when the user clicks on an element	.

3 Copy=The event occurs when the user copies the content of an element.

4 Cut=The event occurs when the user cuts the content of an element.

5 drop=The event occurs when the dragged element is dropped on the drop target.

6 mousemove=The event occurs when the pointer is moving while it is over an element.

7 mouseover=The event occurs when the pointer is moved onto an element, or onto one of its children.

8 mouseout=The event occurs when a user moves the mouse pointer out of an element, or out of one of its children.

9 mouseup =The event occurs when a user releases a mouse button over an element.

10 submit=The event occurs when a form is submitted	.

11 waiting=The event occurs when the media has paused but is expected to resume (like when the media pauses to buffer more data)

~~~
# HTML DOM Event Properties and Methods
~~~
1 Buttons=Returns which mouse button was pressed when the mouse event was triggered.

2 cancelable=Returns whether or not an event can have its default action prevented.

3 loaded=Returns how much work has been loaded.
~~~

# Session History Events
~~~
1 pagehide=	A session history entry is being traversed from.
2 pageshow=	A session history entry is being traversed to.
3 popstate=	A session history entry is being navigated to (in certain cases).
~~~

# Form Events
~~~
1 reset	=The reset button is pressed
2 submit=	The submit button is pressed
~~~

# View Events
~~~
1 fullscreenchange=	An element was turned to fullscreen mode or back to normal mode.

2 fullscreenerror=	It was impossible to switch to fullscreen mode for technical reasons or because the permission was denied.

3 resize=	The document view has been resized.

4 scroll=	The document view or an element has been scrolled.
~~~

# Keyboard Events
~~~
1 keydown=	ANY key is pressed

2 keypress=	ANY key except Shift, Fn, CapsLock is in pressed position. (Fired continously.)

3 keyup	=ANY key is released.
~~~

# Mouse Events
~~~
dblclick=	A pointing device button is clicked twice on an element.

mousedown=	A pointing device button is pressed on an element.

mouseenter[=]	A pointing device is moved onto the element that has the listener attached.

mouseleave=	A pointing device is moved off the element that has the listener attached.

mousemove=	A pointing device is moved over an element. (Fired continously as the mouse moves.)
mouseover=	A pointing device is moved onto the element that has the listener attached or onto one of its children.

mouseout=	A pointing device is moved off the element that has the listener attached or off one of its children.

~~~
